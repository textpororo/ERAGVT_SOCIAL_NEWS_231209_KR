eraGVT(2879.zip版)用の촉수服(テンタクルスーツ)強化＆改造+αパッチです
本体2879.zip版に、eraGVT(2879.zip版)用勝手に追加&修正パッチ1.5(2910.zip)を付属パッチ込みで当ててから使用してください

촉수服関連の変更点
	施設の拡張→촉수研究のLvUPで、テンタクルスーツのカスタマイズを追加
		見た目はデフォルトが体に촉수が巻き付いている＆局部が露出(元からあった지문参照)、他は非暴走時では普通の衣装とほぼ変わらない見た目のイメージです
		能力補正系はバランスブレイカー気味な気もするけど、条件やデメリットが厳しめなので大丈夫…だといいなぁ
	暴走時の調教스테이터스増加が固定値だったのを、能力や素質などで変化するように(おまけで촉수拘束具も)
	暴走時の調教스테이터스増加によって体力、気力などが減少するように(촉수拘束具も)
	変身前後で共通装備強制から、別々に装備できるように(変身時に装備、かつ暴走時は変身解除しないようにしています)

それと追加で変更＆修正した部分
	変身前と変身時で同じ衣装を装備している際に変身前を속옷겸용化していると、変身時は속옷겸용化していない場合でも속옷겸용化状態になるのを修正
	一部の壊れない衣装を装備している場合に、裸の状態と同じ恥情が入らないように調整
	기절中などの場合、自動Ｖ防御が発生しないように修正
	拡張攻め系による촉수の太さ補正が弱すぎたと思ったので強化
	格下相手でもミスがよく出るので、基本の命中上限を95に上方修正
	スレ472の不具合を修正(前に自分がCFLAG:1に2があるのを忘れていて色々いじったせいでした、ごめんなさい)
