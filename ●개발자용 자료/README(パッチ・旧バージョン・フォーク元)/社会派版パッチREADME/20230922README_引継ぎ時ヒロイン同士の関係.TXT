eraGVT_社会派촉수ニュース版230912でのニューゲーム時にできるヒロイン同士の関係設定を
クリア後データ引継ぎした後のキャラ作成後にもできるようにしたパッチです。
使い方は同梱のERBフォルダをeraGVT_社会派촉수ニュース版230912のERBに上書きすればオーケーです。
以下、元ファイルからの修正内容。



ERB\ゲーム内_イベント発生\オープニング処理.ERB
175行目以降にあった컨피그のプリセット設定処理をひとまとめにして呼び出せるように
@HEROINE_PRESETで関数化しファイル下部に移動、元の場所に

CALL HEROINE_PRESET

を追記


ERB\ゲーム内_イベント発生\エンディング\SUCCESSION.ERB
キャラクター引継ぎ諸々の処理が終わっている1590行目に

	;컨피그のプリセット設定
	CALL HEROINE_PRESET

を挿入
