下二つのパッチはお互い干渉しないので片方だけでも両方入れても大丈夫です
・下着のEQUIP番号ずらすパッチ
	Ver1の下着のEQUIP番号部分を切り分けた＆いじり忘れていたところを直したパッチです
	内部データの番号を変えてるだけなので普通にプレイしているだけなら影響ないはずです(속옷のカスタマイズ以外はですが)
	こちらを当てる場合、Ver1以前に속옷をカスタマイズしていた場合、初期化されてると思うのでお手数ですが設定し直しをお願いします

・幽閉、拡張、흑화関係調整パッチ
	拡張を表示필터がオンの場合、幽閉時の調教でも촉수の太さや数を表示するように
	極端な拡張を抑制するがオンの場合、敵の촉수の太さランクがA以上にならないように
	幽閉中に흑화 캐릭터から犯される場合、흑화 캐릭터の番号指定が間違っていたのを修正
	戦闘中の敵がピストン系コマンドを使ってきた際、촉수の太さや数を前ターンのと同じにするように(TFLAG:101～109を使用)
	その他色々調整



本体作者様へ、もし下着のEQUIP番号ずらすパッチを導入されるなら、お手数ですが次回更新の際にでもアップデート処理のところに↓のような処理を入れてもらえるとありがたいです。
FOR LOCAL,1, CHARANUM
	FOR LOCAL:1,300, 400
		IF EQUIP:LOCAL:(LOCAL:1) == 0 && EQUIP:LOCAL:(LOCAL:1 - 100) > 0
			EQUIP:LOCAL:(LOCAL:1) = EQUIP:LOCAL:(LOCAL:1 - 100)
		ENDIF
		EQUIP:LOCAL:(LOCAL:1 - 100) = 0
	NEXT
