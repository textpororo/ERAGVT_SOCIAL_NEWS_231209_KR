eraGVT社会派220121用보추対応흑화夜這いイベントパッチ

★概要
まだまだ보추用にイベント修正中ですが、
あまりに量が多いので先んじて흑화襲撃と夜這いだけ試しに公開してみます。

意図としては修正方針を試して貰うというのと、
デバッグも兼ねて欲しいという目的があります。

最低限のチェックはしているのでエラー落ちすることは（多分）ないはずです。

素晴らしい지문と理解しやすい綺麗な構文を書いてくれた先人に感謝……！

本パッチはeraはおろかプログラミング素人が作成したものです。
品質には注意していますがどこかおかしいところがないとも言い切れません。
というか多分あるので自己責任でバックアップはしておいてください。

【具体的な変更点（憶えている限りの）】
보추は男女平等필터ーの影響を受けません。
ONだろうとOFFだろうと膣がなくペニスがあるという点以外は女扱いを受けます。
三人称でも基本的に意図的に女の子扱いしています。

基本的に既存の지문をコピー改変させて頂いたもので追加はほぼしていません。
（보추のＡ系夜這いをＶ系夜這いの改変にしたりはしてますが）
ただ、Ｂ夜這いだけは相手の욕망が高い場合のパターンが物足りなかったので、
僭越ながら自分が보추で相手の욕망が高い場合だけまるまる追加しました。

また、今後の方針的にも基本的には元々남자対応してないものは、
보추に対応はさせても보추ではない남자に対応はさせないつもりです。
が、흑화襲撃だけは보추ではない남자にも対応させてみました。
もちろん、보추ではない남자인 경우에는 男女平等化필터ーの影響を受けます。

なにかありましたらスレで教えて頂ければ修正します。
今回修正された흑화襲撃や夜這いイベントで、
남자なのにＶ경험이 入った、지문で膣があること扱いにされている、
など些細なことでも教えて貰えると助かります。
