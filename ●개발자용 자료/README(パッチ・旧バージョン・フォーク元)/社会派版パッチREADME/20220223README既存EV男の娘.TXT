eraGVT社会派220213用既存イベント오토코노코対応パッチ

★概要
既存イベントを오토코노코に対応させました。
오토코노코は양성평등필터の影響を受けません。

元々남자に対応していなかったイベントも、
女性器が必要なものでなければ出来る限り오토코노코に対応させています。

ただし、戦闘系の対応はまだ完全ではありません。
（例えば쥐어짠다コマンドで오토코노코は애널で搾り取れるようにしたいのですが、
　戦闘系の処理を解読しきれずに手を出せていません……）

少しですが不具合などもついでに修正しています。
念のため、本パッチは元のファイルをバックアップのあと、適用してください。


★導入方法
以下の環境に上書きしてください。
・eraGVT_社会派촉수ニュース版220213


★具体的な変更点（憶えている限りの）
・男女필터に関係なく오토코노코は犯されるように既存イベントを変更
・오토코노코含む남자でも지문が不自然ではなくなるように分岐を追加
　（三項演算子を多用しているので読みにくくなってしまっているかも……）
・오토코노코のＡセックス関係は一部Ｖセックス関係を流用したものを追加
　（その場合はＡ감각が１以下で처녀と似たような扱いをされます）

・보스 촉수襲撃の際にターン数がおかしくなっていた不具合を修正
・속옷を着替えたときメッセージが表示されなかった不具合を修正
・その他、目についた誤字脱字と思われる部分は修正しています


★やりたいことリスト
・エルデンリング、おおエルデンリング
・息抜きがてらに고집쟁이or순종적구상作成

★やりたいけど難しそうなもの
・戦闘コマンドの오토코노코対応
・오토코노코用後天素質「後孔묘상화（仮）」追加
・それに伴う애널임신
　（どんな生物の精液でも촉수か合いの子を孕み、まともな인간の赤ん坊は孕めない）
