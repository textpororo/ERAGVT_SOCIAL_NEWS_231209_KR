eraGVT社会派220302用既存구상남자対応３種パッチ

★概要
既存の구상、진지함、츤데레、과묵함を남자に対応させました。
既存EV보추対応で抜けがあったので、２つのファイルを修正しました。
衣装の下着保護、挿入抵抗カスタマイズの数値処理が逆転していたと思われるので修正しました。


★導入方法
以下の環境に上書きしてください。
・eraGVT_社会派촉수ニュース版220302


★具体的な変更点（憶えている限りの）
・구상については主に台詞を남자っぽくし、
　地の文を、남자でもオンナでも不自然ではないようにしただけです。
　新たに追加した部分はほぼありません。
　対応した性格のキャラクターで汎用남자구상を選べば使用できます。
　남자っぽい口調の女の子、ということでも使えるかと思います。

・既存EV보추対応で追加したファイルは次の２種類です。
　ENDING.ERB
　MESSAGE_SUBEVENT.ERB
　ほんのちょっとの修正なので大した影響はないと思います。

・衣装の挿入抵抗カスタマイズの数値処理の修正については、
　衣装固定カスタマイズ.txtにて、「値が低いほど下着保護力が高くなる」とあるので、
　おそらくその辺りが原因で計算ミスが生じたと判断しました。
（修正といっても一箇所、+=を-=に変えただけです）


★やりたいことリスト
・エルデンリングで囚人服を着て駆け回る

・촉수 구속구の強制装着の機会を増やす
　撤退時にも強制装着の機会を作る
　撤退成功すると伏兵촉수に不意打ちを食らう（방위력で35%～85%に変化）
　回避判定で失敗すると衣装耐久度が残っていても強制装着
　衣装耐久度が元々減っている場合は上記に加えて時間切れ時と同じ確率で強制装着
　時間切れ時より強制装着されやすくすることで撤退のリスクを高めてみたい

・촉수 구속구を外し難くする
　촉수の欠片２個と5000$くらい必要にして、しかも確率で失敗するように
　失敗確率は욕망と촉수중독の레벨と、음란と촉수의포로の有無で変化
　１레벨ごとに5%、음란と촉수의포로で10%ずつ減少くらい？（解除成功率90～20%）
　絶対外したいときはセーブ＆ロードできるからいいよね、くらいの気持ちで……

・長期戦モード（仮）を追加する
　刻印や性的스테이터스の成長が５分の１？の速度になる
　代わりに被ダメージと与ダメージが雑に半分になる（各種回復量は変化なし）
　최종 보스 촉수が無理ゲーになる可能性あり

・촉수 구속구の強化も、長期戦モードも、컨피그で設定可能に
