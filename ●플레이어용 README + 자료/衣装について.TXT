■衣装について
衣装は겉옷（외관の服）と속옷（下着類）に分かれる。
변신능력を持つキャラは変身前の겉옷と変身後の겉옷を個別に設定できる。
속옷は変身前・変身後とも共通。
変身後のみしか着ることの出来ない衣装も存在する。
衣装はそれぞれ耐久度が設定されており、촉수の공격によって徐々に破壊される。
ただし、衣装は破壊されても消失せず、戦闘終了後には必ず完璧に復元される。
衣装には스테이터스補正や特殊な効果を有するものもある。
衣装による補正や効果は衣装が破壊されると効果を発揮しない。

　■衣装の耐久度と破壊について
　　衣装の破壊は、まず겉옷からダメージを受ける。
　　겉옷の損傷度が50%を切ったら、속옷もダメージを受けるようになる。
　　β１までのように、変身解除→再変身で損傷度は回復しないので注意。
　　当然、속옷の損傷度は変身前から引き継ぐ。
