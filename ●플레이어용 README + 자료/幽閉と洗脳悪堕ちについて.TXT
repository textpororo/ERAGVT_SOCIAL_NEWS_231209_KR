■유폐について
보스 촉수、최종 보스 촉수との戦闘に敗北した場合、捕らわれの身となって毎ターン終了時に調教されます。
戦闘時に行われる性的コマンドより強力かつ、特殊なものが多くなっています。
捕らえられたキャラは、他のキャラが捕らえ主の촉수を倒すことで구출한다ことができます。
長く捕らわれた状態にあるキャラはやがて取り込まれ、死亡扱いとなります。
また、苛烈な調教によりキャラが세뇌され、あるいは흑화して以降敵として現れるようになります（下記に詳細）。

■유폐中の調教内容について
・C中心攻め                  클리토리스を中心に攻めます。ヴァギナや애널は未経験なら攻めません。
・V中心攻め                  ヴァギナを中心に攻めます。애널は未経験なら攻めません。
・A中心攻め                  애널を中心に攻めます。ヴァギナは未経験なら攻めません。
・B中心攻め                  バストを中心に攻めます。ヴァギナや애널は未経験なら攻めません。
・Sな攻め                    サドな攻めです。고통쾌락경험이 入る場合があります。
・強制奉仕                   強制的に奉仕をさせる攻めです。봉사쾌락경험이 入る場合があります。
・羞恥な攻め                 羞恥心を煽る攻めです。노출쾌락경험이 入る場合があります。
・후타나리化/후타나리攻め    キャラを후타나리にします。既に후타나리の場合は快Cを多く与えます。Ｃ촉수とK촉수のみ使用してきます。
・V拡張                      ヴァギナを拡張します。ヴァギナが未経験ならV中心攻めになります。V촉수とK촉수のみ使用してきます。
・A拡張                      애널を拡張します。애널が未経験ならA中心攻めになります。A촉수とK촉수のみ使用していきます。
・모유체질化/搾乳攻め        キャラを모유체질にします。既に모유체질の場合は快Bを多く与えます。B촉수とK촉수のみ使用してきます。
・Vに産卵                    ヴァギナに産卵します。ヴァギナの拡張経験が少ない場合はV拡張になります。V촉수とK촉수のみ使用してきます。
・Aに産卵                    애널に産卵します。애널の拡張経験が少ない場合はA拡張になります。A촉수とK촉수のみ使用してきます。
・注射                       キャラに注射します。他の調教に比べ、キャラの汚染度が高く、早く세뇌されてしまいます。条件を満たすことで음란系素質の取得トリガーになる場合があります。
・촉수の기생		　　 キャラに촉수が기생します。他の調教に比べ、キャラの汚染度が非常に高く、早く세뇌されてしまいます。
・새끼 촉수に犯される	　　 自分の産んだ새끼 촉수に犯されます。임신出産機能がオンで새끼 촉수を出産した場合のみ発生します。

■세뇌と흑화について
세뇌/흑화状態になるには、유폐中の調教で与えられる汚染度が影響します。
キャラの取り込みが発生する前に、一定以上の汚染度に達すると、세뇌/흑화が発生します。
イベント発生時に、촉수의포로を持っていると흑화に、持っていないと세뇌になります。
세뇌/흑화したキャラは以後、敵キャラとしてエンカウントするようになります。
세뇌状態にあるキャラは、直接本人を倒すか捕らえた촉수を倒すことで解放されます。
흑화状態にあるキャラは、直接本人を倒すことで解放されます。

세뇌キャラが味方側のキャラを倒した場合は、その주인の촉수が代わりにキャラを유폐します。
흑화 캐릭터가 味方側のキャラを倒した場合は、흑화 캐릭터가 촉수に기생されていればキャラを유폐し、기생されていなければキャラを一定量犯して解放します。



(詳細仕様@社会派版)
	①흑화オプションがOffの場合、유폐から20ターン(10日)後に取り込まれ死亡する
	②흑화オプションがOnの場合、一定の日数経過後に세뇌される。촉수의포로持ちなら흑화する。
	　※ただし「묘상화で産む機械に」オプションがOnなら、[묘상화]取得キャラに限って取り込まれが発生する
	③[희롱특화체질]持ちは絶対に取り込まれず、흑화しないオプションがOnなら흑화もしない(無敵！)
	④[非戦闘員]持ちは現在촉수に襲撃されず、したがって유폐もされない
