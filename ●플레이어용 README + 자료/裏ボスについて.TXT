
■히든 보스との遭遇方法

　周回累計数１回以上で難易度HARDCOREの最後に出現する。
　攻略法を熟知していればHARDCOREの최종 보스を撃破した直後でも勝てる。
　諦めずに공격を続けていくことが大事。

第一形態　천사の樹

硬い外殻に守られた、非常に高い방어力を誇る形態。
パラメータを공격力に極振りした脳筋キャラでもダメージが非常に通りにくい。

実は천사の樹の방어力はキャラクターの공격力に応じて上昇しており、
さらに공격をガードする（カス当たりにする）特殊能力まで持っているため、
いくら공격力を伸ばしてもダメージが通らないようになっている。

非常に厄介な特殊能力だが、
実は「천사の樹が前のターンに공격を行った距離」はガード能力が働かない弱点になっているので、
正攻法としては相手の動きに合わせて「自分の공격は通り、相手の공격は届かない距離」を選んで
ダメージを与えていくのが基本となる。

また、「아무것도안한다」を選んであえて捕まり、コマンド「날뛴다」で방어力無視のダメージを与えるという手もある。
사정させて油断を誘えば방어力が低下するので、その隙を突いて공격すればダメージが通りやすい（날뛴다で十分かもしれないが）。
ただし천사の樹は非常に搾精ダメージが通りにくいため、사정が早くとも絞り殺すのは困難である。

천사の樹の체력を７５％まで削ると第二形態になる。





第二形態　楽園の花

妖しい香りで相手を惑わす、トリッキーな形態。
방어力が大きく落ちている上に特殊能力を失っているためダメージを与えるのに苦労はしないが、
지성が異常に高く、공격予測が働かない厄介な形態になっている。

この形態では체력と기력のうち消耗の激しい方を追い打ちしてくる習性があるので、
방어力の高いキャラの체력をわざと減らしておいて耐えながら공격するという手がある。

この形態からは방어無視、기력に大ダメージの「邪悪な波動」を撃ってくる。
기력の少ないキャラはあっという間に変身を解かれてしまう危険があるので、
その場合は冷静に距離を取り、再び変身できるだけの기력を確保しておきたい。

楽園の花の체력を３７％まで削ると第三形態になる。





第三形態　타락の核

非常に高い공격力と민첩性を持つ공격的な形態。
直撃を食らうと방어に振ったキャラでも大きく체력を削られてしまう。
変な能力は持っていないがパラメータの高さで圧倒してくる。

민첩性に特化したキャラでも命中回避を安定させるのは難しいため、
근거리型ならば命中率の保証40%に賭けるか、
中、원거리型ならカス当たりダメージに期待して체력を高めに維持する必要がある。

特に相手の원거리が手薄になったターンに距離を取ってこまめに回復するのが大事。
高い민첩性を持つとはいえ、공격の届かない距離に居れば安全ということは変わらないので、
基本に立ち返って挑もう。

타락の核の체력を１０％以下まで削ると체력を７０％まで回復してくるが、
この段階に入ると방어力と민첩性が大きく落ちているのでキュアプ○ムスほど절망する必要はない。
そのまま一気に畳みかけて、引導を渡してやろう。
