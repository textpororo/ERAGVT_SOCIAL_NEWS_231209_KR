

種族とフィート
	それぞれの種族には６つのフィート（特徴）が설정されており、種族決定時に最大で３つまで選択することができる。
	キャラメイク時に種族を選択せずにランダムに指定した場合は、フィートもランダムで決定される。
	また、フィートを１つも選択しないことも自由である。

	なお、種族ごとに固有フィートが存在するが、固有フィートの方が高性能というわけではなく
	その種族でしか実現できない組み合わせによって差別化を図っている。



인간
　　突出した長所も欠点もないオーソドックスな性能。
　　フィートは性能よりもロールプレイ重視のラインナップとなっているが、
　　[축복]と[불굴]を併用できるので安定性は高い。

	평범
	말려드는체질
	숨겨진힘
	행운아
	축복
	불굴

異能者
　　特殊な能力を持った인간。
　　シリアスな雰囲気のフィートが多くバトル漫画の登場人物のようなキャラを作ることができる。
　　直撃率を高める[공세구축][심안][사냥꾼의직감]を重ねられるので공격面を安定させやすい。

	공세구축
	숨겨진힘
	타고난전사
	핫스타트
	사냥꾼의직감
	심안

마법사
　　魔法を身に付けた인간。
　　EX 게이지増強、火力強化、後方支援、疲労サポートと何でもござれの万能型。
　　どの組み合わせでも違った強みを発揮するが持久力だけは補えない。

	마력저장
	행운아
	공중부유
	풀버스트
	불로장수
	지혜의극치

인조인간/폴리니안
　　機械の身体を持った存在。폴리니안は인간と인조인간の中間で子供も普通に産める。
　　全体的に高性能だがデメリットが目立つフィートが多い。
　　非常にクセの強いキャラクターになりやすいので、あえてフィートを少なめに抑えるという選択肢もある。

	고속회피
	근력강화
	공중부유

	풀버스트
	스태미나

천사
　　邪を滅する気高き천사。くっころ属性持ちの高パラメータ種族。
　　[날개]と[공중부유]を両方取得すれば気兼ねなく空中戦を行える。
　　[신기소유자]で戦闘向けに、[지혜의극치]で後方支援向けの性能に変化する。

	신기소유자

	공중부유
	축복

	배덕의낙인

마족
　　戦いを好む闇の眷属。くっころ属性持ちの高パラメータ種族。
　　공격に特化することが可能だが、その分だけ小回りが利かないキャラになっていく。
　　[불굴]によって高い공격性能を維持できるので強気に攻め続けることができる。

　　好きなようにフィートを盛っても扱いやすいのが利点。

 ==== 固有フィート ====

	・평범
		あなたは目立つところがなく평범だ。少なくとも、以前まではそのはずだった。
		　利点：拘束された際に敵を油断させやすい。
		　代償：매료경험이 上昇しにくい。

	・말려드는체질
		あなたは何故かトラブルに巻き込まれやすい。それはあなたが他人の不幸を放って置けない"お人好し"だからかもしれない。
		　利点：出撃時に보스 촉수が出現しやすくなる。
		　代償：襲撃イベントのターゲットにされやすい。観衆に遭遇しやすい。

	・공세구축
		あなたは공격に向いた能力を持っているか、本来は戦闘向きでない能力を공격に転用することに長けている。
		　利点：공격の直撃率が6%上昇する。
		　代償：拘束された際に敵を油断させにくい。

	・마력저장
		あなたは非常時に備えて日頃から魔力を蓄えている。
		　利点：EX 게이지が２つ溜まった状態で戦闘を開始する。
		　代償：休憩時の回復量が少し低下する。

	・고속회피
		あなたは通常ではあり得ない反射速度で物事に対処することができる。目の前に迫った危機もあなたにとっては止まっているに等しい。
		　利点：戦闘中に一度だけ、被弾を無効化して공격を絶対回避することができる。
		　代償：공격ボーナスが-5される。

	・신기소유자
		あなたは天より授けられた神器を所有している。神器は持ち主を護り、身体能力を強化する機能を持つ。
		　利点：공격、방어、민첩ボーナスが+5される。理性強度が+1される。
		　代償：성내성ボーナスが-10され、세뇌、흑화状態になりやすくなる。

	・투쟁본능
		あなたには闘争を好む마족の性質が色濃く受け継がれている。あなたは無意識に戦いを望んでしまう。
		　利点：공격ボーナスが+10され、황홀状態にならなくなる。戦意強度が+1される。
		　代償：성내성ボーナスが-5され、세뇌、흑화状態になりやすくなる。

	・넘치는생명력
		あなたは常に活力に溢れており頑強である。多少のダメージなどものともしない。
		　利点：戦闘中と休憩時の回復量が増加する。
		　代償：지성ボーナスが-10される。

	・정령교신
		あなたは自然界に存在する目に見えない精霊の声を聞くことができる。大地と風の恵みがあなたの味方をするだろう。
		　利点：先制率が上昇し、지성ボーナスが+10される。戦意強度が-1される。
		　代償：공격、방어、민첩ボーナスが-5される。

	・야수의후손
		あなたは獣の性質をその身に色濃く受け継いでおり、身体の一部にその特徴が強く顕れている。
		　利点：クリティカル率が+6%される。인기도가 上昇しやすい。理性強度が-1される。
		　代償：지성ボーナスが-10される。임신しやすい。

	・밤악마의귀족（固定取得）
		あなたは마족の中でも特に高貴な、夜を支配する뱀파이어の一員である。伝承通りの強大な力と、日光を苦手とする弱点を持っている。
		　利点：全能力ボーナスが+10される。また、昼間の休憩時の回復量が上昇する。
		　代償：昼間に鍛錬、休憩、전투 지원以外の行動を選択できない。

 ==== 汎用フィート ====

	・숨겨진힘
		あなたは普段から人より劣って見られがちである。しかし、それは真の実力を発揮できていないだけなのだ。
		　利点：체력, 기력が残り25%以下のときクリティカル率が12%上昇する。
		　代償：공격、방어、민첩、지성ボーナスが-5される。

	・행운아
		あなたはとても幸運に恵まれている。もっとも、運が良すぎるせいで実力をあまり評価して貰えないのだが。
		　利点：특별 활동、정보 수집で良い結果になりやすい。
		　代償：공격、방어、민첩、지성ボーナスが-5される。

	・타고난전사
		あなたは過酷な生い立ちのためか信念のためか、戦う以外の生きる道を知らない。
		　利点：敵にクリティカルヒットされず、타격공격では기절しない。理性強度と戦意強度が+1される。
		　代償：특별 활동と전투 지원に参加できなくなる。

	・근력강화
		あなたの腕力はとても強い。力尽くであれば、大抵の物事には負けない自信がある。
		　利点：공격ボーナスが+5され、「풀어낸다」の成功率が上昇する。
		　代償：「풀어낸다」を実行すると疲労が蓄積する。

	・날개
		あなたは背中に翼を持っており、その生まれ持った能力を扱うことに長けている。
		　利点：방어ボーナスが+5される。공중대시ゲージが１つ増加する。
		　代償：着地時に疲労が増加する。

	・공중부유
		あなたは魔法か科学か念力か、何らかの方法で宙に浮いたまま移動ができる。
		　利点：공중대시使用時に疲労が蓄積しない。
		　代償：移動系コマンドの疲労蓄積が1増加する。

	・소형체구
		あなたは種族的に小さな体格をしている。非力な分、身軽で隙のない動きができる。
		　利点：민첩ボーナスが+5される。공중대시使用後の着地ペナルティを無視できる。
		　代償：「풀어낸다」の成功率が低下する。

	・핫스타트
		あなたは自分自身の能力を熟知しており、最初から全力を出して戦うことができる。
		　利点：EX 게이지に関わらず常にバースト공격が使用可能になる。
		　代償：バースト공격使用時の疲労蓄積が1増加する。

	・풀버스트
		あなたは能力を一時的に高めることで強烈な一撃を放つことができる。
		　利点：バースト공격で与えるダメージが上昇する。
		　代償：バースト공격使用時の疲労蓄積が1増加する。

	・축복
		あなたは目に見えない何らかの力に守護されている。その恩恵で、子宝にも恵まれるであろう。
		　利点：あらゆる状態異常になりにくい。理性強度が+1される。
		　代償：粘液공격、波動공격で受けるダメージが少し増加する。임신しやすい。

	・불굴
		あなたは何があろうと決して諦めない。最大の敵は自分自身の弱さであることを、あなたは知っているからだ。
		　利点：체력、기력を消耗しても行動成功率が低下しにくい。戦意強度が+1される。
		　代償：EX 게이지が増加しにくい。

	・스태미나
		あなたは優れた持久力を持っている。多少の疲労ではへこたれない。
		　利点：疲労が蓄積しても回復行動の回復量が低下しにくい。
		　代償：EX 게이지が増加しにくい。

	・불로장수
		あなたは定命の存在でありながら老いることがない特異体質である。
		　利点：休憩しなくても疲労が少しずつ回復する。
		　代償：休憩時の回復量が低下する。임신しにくい。

	・지혜의극치
		あなたは豊富な知識と、その知識を生かす頭脳を持ち合わせている。
		　利点：지성ボーナスが+10される。
		　代償：체력、기력、성내성ボーナスが-5される。

	・사냥꾼의직감
		あなたは獲物の居場所を探り当てるセンスに優れている。そして、時には勘に頼ることも重要だと思っている。
		　利点：先制率が上昇し、잡몹적撃破時に探索度が多めに上昇する。
		　代償：지성ボーナスが-5される。

	・심안
		あなたは非常に鋭い감각を持っている。その能力を生かして、あなたは敵の弱点を的確に공격することに長けている。
		　利点：直撃率とクリティカル率が3%上昇する。
		　代償：방어、민첩ボーナスが-5される。

	・남다른미모
		あなたはとても神秘的な、または妖艶な魅力を備えている。あなたがその気になれば、他人を魅了するなど容易いことだろう。
		　利点：인기도と매료경험이 上昇しやすい。
		　代償：襲撃イベントのターゲットにされやすい。観衆に遭遇しやすい。

	・배덕의낙인
		あなたは事実であるかどうかに関わらず、その身に"背徳者"の汚名を着せられている。その汚名に屈するかどうかはあなた次第だろう。
		　利点：搾精공격で与えるダメージが上昇する。
		　代償：戦闘や유폐時に獲得する珠の量が増加する。

	・에어마스터
		あなたは素早い身のこなしにより空中での機動力に長けている。空中であなたを捉えることは至難の業だろう。
		　利点：공중대시中の回避率が25%上昇する。
		　代償：공중대시使用時の疲労蓄積が1増加する。
